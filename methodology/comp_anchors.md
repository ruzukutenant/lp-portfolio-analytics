# How to use this table

1. Identify the position's category (or closest analogue)
2. Pick a bull anchor, a base anchor, and a bear anchor from the relevant rows
3. Set scenario probabilities reflecting:
   - How much the company resembles each comp
   - Recent multiple-compression direction
   - AI tailwind strength (see `ai_tailwind_assumptions.md` in this directory)
4. Compute weighted multiplier × current dilAdj × DF for expected exit

When in doubt, the safer path is anchoring to the base or bear comp (multiple compression risk is real), unless the company has an extraordinary growth signal that justifies the bull anchor.
